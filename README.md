# renovate-config

A [shareable config preset for Renovate](https://docs.renovatebot.com/config-presets/) used in [my repos](https://github.com/417-72KI?tab=repositories).

```json
{
  "extends": [
    "github>417-72KI/renovate-config"
  ]
}
```
